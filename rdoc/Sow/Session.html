Session

Session provides a central store of commandline options and system state related to invocation the sow command. This includes the destination directory, it’s metadata and any special sow configurations.

36:     def load_raw_pom_metadata

39:       if destination.glob('PROFILE').first
40:         begin
41:           profile = YAML.load(File.new('PROFILE'))

53:           end
54:         rescue
55:         end
56:       end
